================================================

THE PROBLEM:
Your .env file has placeholder values instead of real API keys.

THE SOLUTION (2 Minutes):
==========================

STEP 1: Get FREE Gemini API Key
---------------------------------
1. Open browser: https://aistudio.google.com/
2. Sign in with Google account
3. Click "Get API Key" (top right)
4. Click "Create API key"
5. Copy the key (looks like: AIzaSyD...)

STEP 2: Edit Your .env File
-----------------------------
1. Open: backend\.env in notepad
2. Find this line:
   GEMINI_API_KEY=your-gemini-api-key

3. Replace with your real key:
   GEMINI_API_KEY=AIzaSyD_YOUR_ACTUAL_KEY_HERE

4. Save the file

STEP 3: Restart Backend
------------------------
1. Go to backend terminal
2. Press Ctrl+C to stop
3. Run: python main.py
4. Wait for "Application startup complete"

STEP 4: Test It
----------------
Open new terminal and run:
curl "http://localhost:8000/api/idea?topic=biology"

You should see a JSON response with a project idea!

THEN: Go to http://localhost:5173/try and generate projects!
